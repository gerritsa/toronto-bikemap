# Toronto Bike Share Flow Visualizer

Production MVP for animating Bike Share Toronto 2026 trip flows on a MapLibre/deck.gl map.

## Production Architecture

- `app/` is a Vite React app deployed to Vercel.
- `pipeline/` downloads Toronto Open Data trips, normalizes station/trip records, calls local OSRM to precompute bicycle route geometries, writes Parquet artifacts, and publishes those artifacts to Cloudflare R2.
- Cloudflare R2 hosts generated app data: manifest, trips, routes, and stations.
- The deployed app does not call OSRM. OSRM is only used during local/monthly preprocessing.
- The default production basemap uses CARTO CDN through MapLibre. PMTiles/Protomaps remains optional and can be added later by publishing a `basemapUrl`.

See [docs/data-and-ops.md](docs/data-and-ops.md) for the full production runbook.

## Layout

- `app/` - Vite React app for Vercel.
- `pipeline/` - Python data pipeline for CKAN ingest, station normalization, OSRM routing, Parquet output, QA, and R2 publishing.
- `pipeline/manual_overrides/stations.json` - retired station coordinate overrides needed when historical trips reference stations missing from current GBFS.
- `docs/` - operating notes, data sources, deployment, and licensing.

## Local App

```bash
pnpm install
pnpm dev
```

By default the app loads the included sample manifest from `app/public/sample/latest.json`.

To test against production R2 data locally:

```bash
VITE_MANIFEST_URL=https://<r2-public-host>/manifest/latest.json pnpm dev
```

## Pipeline Setup

```bash
python3 -m venv .venv
. .venv/bin/activate
pip install -r pipeline/requirements.txt
PYTHONPATH=pipeline python -m bikeshare_pipeline.run --help
```

## Production Deploy

Vercel should use:

```text
Root directory: app
Build command: pnpm build
Output directory: dist
Environment variable:
  VITE_MANIFEST_URL=https://<r2-public-host>/manifest/latest.json
```

Cloudflare R2 must host `manifest/latest.json` and the immutable `runs/{run_id}/...` artifacts generated by the pipeline.

## Verification

Run these before deploying app or publishing new pipeline changes:

```bash
pnpm build
PYTHONPYCACHEPREFIX=pipeline/.pycache .venv/bin/python -m compileall pipeline
PYTHONPATH=pipeline .venv/bin/pytest pipeline/tests
```
